=========INICIAR APLICACION WEB=========

Para iniciar la aplicación Web se tiene que ejecutar el archivo "run.bat".

> AVISO: Tienes que tener todas las dependencias instaladas para que funcione de forma correcta.

=========ACTUALIZAR DDBB=========

En tal caso de que quieras actualizar la DDBB de la misma aplicación por esta de aquí, tendrás que depositarla en la carpeta llamada DDBB_UPDATE y ejecutar el "update-db.bat" para que se haga el proceso de forma automática (Pero tiene que tener la misma estructura de DDBB que la que se esta utilizando).

=========DEPENDENCIAS=========

Para instalar todas las dependencias, todo estará en la carpeta llamada "Dependencias".
